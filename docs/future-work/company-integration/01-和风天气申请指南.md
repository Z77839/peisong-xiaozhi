## 📋 申请步骤

### 步骤 1：注册（2 分钟）
- 访问 https://dev.qweather.com/
- 点 **注册** → 选 **个人开发者**
- 填手机号 + 验证码 + 密码
- 完成实名认证（要身份证号）

### 步骤 2：创建应用（1 分钟）
- 登录后 → 左侧 **控制台** → **应用管理**
- 点 **创建应用**
- 填：应用名 `配送小智`、应用类型 `Web API`（或免费订阅）
- 提交

### 步骤 3：创建凭据（1 分钟）
- 进入应用详情
- 点 **创建凭据**
- 选 **API Key** 方式
- 名称填 `后端`
- 创建后复制 **API Key** 和 **专属 Host**

### 步骤 4：勾选 API
- 凭据详情 → **启用 API**
- 勾选需要用到的：
  - 天气实况
  - 7 天天气预报
  - 24 小时预报
- 保存

### 步骤 5：配到 Render
1. https://dashboard.render.com/ → peisong-backend → Environment
2. 加：
   - `QWEATHER_API_KEY` = `你的key`
   - `QWEATHER_API_HOST` = `你的专属host.re.qweatherapi.com`（**不加** https://）
3. 保存 → 自动重启

### 步骤 6：验证
```powershell
Invoke-WebRequest "https://peisong-backend.onrender.com/api/adapters/weather/forecast?city=hengyang" -UseBasicParsing
```

应该看到 `"source": "qweather"`。

## ⚠️ 注意事项

1. **1-2 小时生效**：创建后 1-2 小时才能调通
2. **每日配额**：免费版 1000 次/天
3. **Host 唯一性**：每个 Key 对应专属 Host，不能用 devapi.qweather.com
