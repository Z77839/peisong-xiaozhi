## 🛡️ 数据安全

1. **传输加密**：所有数据走 HTTPS / WSS
2. **存储加密**：敏感字段（手机号 / 地址）AES-256 加密
3. **权限控制**：按角色（admin/operator/analyst）严格限制
4. **审计日志**：所有操作留痕，可追溯
5. **数据脱敏**：测试环境自动脱敏消费者信息
